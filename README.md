# Mortgage Calculator

A responsive mortgage calculator built with React, Vite, and Tailwind CSS. It estimates monthly payments with principal and interest, property tax, home insurance, HOA fees, and PMI. It also includes a first-year amortization view, FAQ content, and basic SEO components.

## Features

- Monthly mortgage payment estimate
- Principal and interest breakdown
- Property tax, insurance, HOA, and PMI support
- First-year amortization schedule
- SEO-friendly React single-page setup
- Ready for GitHub + Netlify deployment

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Netlify settings

- Build command: `npm run build`
- Publish directory: `dist`
- Node version: `20`

The included [`netlify.toml`](./netlify.toml) already matches these settings.
